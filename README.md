## Getting Started

1. Clone this repo

   ```sh
   git clone https://github.com/fahmiidris-labs/fahmiidris.dev.git
   ```

2. Go into the project root directory

   ```sh
   cd fahmiidris.dev
   ```

3. Install JS dependencies

   ```sh
   yarn
   ```

4. Start the dev server

   ```sh
   yarn dev
   ```
