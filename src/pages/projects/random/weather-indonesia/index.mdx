## What's this website?

This is a website that is the result of learning to create a Weather Forecast website using the public API of the **Open Weather Map** API.

---

## Permissions

To get weather forecast data, the browser must allow the website to access the location. If not allowed, then the weather forecast data will not appear.

---

## Daily weather forecast and search your city

There are 2 main features, namely weather forecasts for the next 8 days and weather forecasts based on the name of the city we are looking for in the search field.
